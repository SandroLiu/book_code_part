##父子控制器

当一个子控制器添加到父控制器上会自动调用`willMoveToParentViewController:`方法，同时需要子控制器调用`didMoveToParentViewController:`参数为父控制器

当一个子控制器从父控制器移除时，移除前调用`willMoveToParentViewController:nil`告诉子控制器其将要被移除，当从父控制器移除后调用`removeFromParentViewController`移除子控制器
